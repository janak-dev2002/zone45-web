## 4. Current Findings

### WARNING — React Hydration Mismatch
**Description:** The console reports several Minified React errors (#418, #423, #425) related to hydration mismatches.
**Impact:** Minor. React falls back to client-side rendering. The site remains functional, but the performance benefit of SSG is slightly reduced during initial load.
**Recommendation:** Investigate discrepancies between the static HTML generated by `vite-react-ssg` and the initial client-side render (often caused by date formatting, window-dependent logic, or browser extensions).

### MINOR — Missing 'name' attributes on inputs
**Description:** Inputs in the contact form and login form are missing the standard `name` attribute.
**Impact:** Minor friction for automated tools/password managers, but does not break the site for users as they use `id` or `placeholder` for interaction.
